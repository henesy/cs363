How the datasets are prepared
 

Summary: The dataset for OOXQuery is prepared manually. The rest of the datasets are prepared by executing a batch in CyDIW batch CyDIW_batch_for_creation_of_datasets.txt at a single click in CyDIW. This helps in keeping the datasets internally consistent. The datasets are automatically directed to appropriate files by CyDIW.

Loading data in an SQL database

UniversityDataset.xml is loaded in and SQL database. For this there is a load command supported in CyDIW that uses "$MySQL:" prefix. The reason for this is so that it can use MySQL connection. This command should eventually become a "$CyDB:" command.

Loading the data for XQuery

The XML documents Peson.xml, Faculty.xml, Student.xml, Course.xml, Offerring.xml, and Enrollment.xml can be used directly from XQuery.

Loading the data in OOXQuery

UniversityOODB.xml and UniversityOODB.xsd can be used directly by OOXQuery as well as XQuery. The query engine has to be schema aware. This is true of the Enterprise version of Saxon query engine for which we renew the license every fiscal year (July 1 to June 30). This is done by SSG and Abigail.
